# Homebrew Tap for LireFlow

This is the official Homebrew tap for [LireFlow](https://github.com/m0etez/LireFlow), a modern, native macOS RSS reader built with SwiftUI.

## Installation

```bash
brew tap m0etez/lireflow
brew install --cask lireflow
```

Or install in one command:

```bash
brew install --cask m0etez/lireflow/lireflow
```

## Updating

```bash
brew upgrade lireflow
```

## Uninstalling

```bash
brew uninstall --cask lireflow
```

To completely remove all data:

```bash
brew uninstall --zap --cask lireflow
```

## About LireFlow

LireFlow is a modern, native macOS RSS reader featuring:

- Clean, distraction-free reading experience
- Dark/Light mode with liquid glass effects
- 200+ default feeds across 25+ categories
- Archive and organization features
- Performance optimization controls
- Weather widget integration
- Full keyboard navigation with vim-style shortcuts

Requires macOS 14.0 (Sonoma) or later.
